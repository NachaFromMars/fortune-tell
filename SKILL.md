# Fortune Tell Experts — Multi-System Fortune Telling

## Mô tả
Claude Code Skill — 4 hệ thống mệnh lý độc lập (Bát Tự, Tử Vi, Kỳ Môn, Mai Hoa) xếp bàn riêng, giải đọc riêng, cross-verify → chỉ xuất kết luận đa số nhất trí.

## Trigger Keywords
`bát tự`, `tứ trụ`, `fortune tell`, `xem mệnh`, `算命`, `八字`, `four pillars`, `multi-system fortune`, `expert panel`, `mệnh lý tổng hợp`

## Cách dùng
Skill hoạt động như expert panel — 4 hệ thống độc lập:
1. **Bát Tự (四柱)** — Tứ Trụ thiên can địa chi
2. **Tử Vi Đẩu Số** — 12 cung + sao
3. **Kỳ Môn Độn Giáp** — Thời vận
4. **Mai Hoa Dịch Số** — Quẻ dịch

Mỗi hệ thống xếp bàn + giải đọc → cross-verify → chỉ xuất kết luận majority vote.
